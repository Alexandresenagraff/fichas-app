# Sons de notificações

Adicione nesta pasta os quatro arquivos de áudio abaixo, em MP3:

- `new-art.mp3`
- `change-request.mp3`
- `art-approved.mp3`
- `urgent.mp3`

O aplicativo já referencia esses nomes e reproduz os sons apenas para notificações novas recebidas em tempo real. Enquanto os MP3 não forem adicionados, ele usa um toque nativo de reserva para cada tipo de evento.
